enska by enska-core-manager

EnsKa Core Manager is a development, deployment and management plateform for your PHP applications.

This plateform was created to elaborate complex and long-life web applications as simple as possible. Top 10 features : * Multi-frameworks integration (EnsKa Core, Symphony, Zend, CodeIgniter ...) * Multi-plateforms authentification system (LDAP, MySQL, WLID, OpenID, Facebook, ...) * Simple Import / Export websites and applications arhives * System backup * Simply WEB Services deployment * Native multi-site * SEO * Cache manager * Roles manager * Form rules checker

Install

push this directory in your apache directory, lauch a navigator on the index and follow the instructions.
